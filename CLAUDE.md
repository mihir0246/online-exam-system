Project name: Online Examination System

This is an online examination system that enables administrators to create and manage test papers, questions, and results; trainees to register and take exams; and trainers to oversee performance and feedback. It solves the problem of efficient remote assessment and evaluation for educational institutions and training programs.

Stack
- Frontend: React
- Backend: Node.js with Express
- Database: MongoDB
- File Storage: AWS S3
- Hosting: Heroku
- Authentication: Passport.js
